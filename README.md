"# Telegram Genealogy Bot\n\nTelegram-бот для управления генеалогическим древом и отслеживания дней рождений.\n\n## Установка\n1. Клонируйте репозиторий:\n   ```bash\n   git clone https://github.com/Glazzeg/telegram-genealogy-bot.git\n   ```\n2. Установите зависимости:\n   ```bash\n   pip install -r requirements.txt\n   ```\n3. Создайте файл `.env` с токеном бота и ID администраторов (см. `.env.example`).\n4. Поместите ваш собственный файл `drevo.ged` в корневую папку проекта (бот ожидает файл с именем `drevo.ged`).\n5. Запустите бота:\n   ```bash\n   python main.py\n   ```\n\n## Использование\n- Администраторы: добавление/удаление пользователей, обновление базы, просмотр дней рождений.\n- Пользователи: просмотр ближайших дней рождений.\n\n## Примечание\nФайл `drevo.ged` не включен в репозиторий, так как он содержит персональные данные. Пользователи должны предоставить свой собственный GEDCOM-файл, названный `drevo.ged`, и поместить его в корень проекта."
